# Multi-Tenant Task Manager SaaS

## Description
A full-stack task management application where multiple teams can create workspaces, projects, and tasks.  
Built with Node.js, Express, MongoDB, React, and Tailwind.  

This project demonstrates authentication, authorization, role-based access, and clean full-stack architecture.



## Features
- User registration & login with JWT authentication
- Role-based access (workspace owner vs member)
- Create workspaces, projects, and tasks
- Assign tasks to members
- Update task status (todo, in-progress, done)
- Clean UI with dashboard template


## Tech Stack
**Frontend:** React, Tailwind CSS, Axios  
**Backend:** Node.js, Express.js, MongoDB, Mongoose  
**Auth:** JWT, bcrypt  
**Deployment:** Vercel (frontend), Render/Railway (backend)
